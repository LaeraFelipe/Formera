formera-form - v1.0.27

Formera

Formera is a form lib focused on high performance and participle. All its functionalities were developed aiming at the smallest size and smallest impact on performace.

Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

Prerequisites

Formera has no dependencies. All its features were developed internally.

Installing

 
npm install --save formera-form@latest

Demo

formera-form demo.

Documentation

The full documentation can be found at formera-form.
